El hardware utilizado para la realización de los experimentos corresponde a un computador de escritorio equipado con un procesador AMD Ryzen™ 7 5700X, el cual cuenta con 8 núcleos y 16 hilos, y una frecuencia base de 3.4 GHz. Además, el sistema dispone de 32 GB de memoria RAM DDR4 a 3200 MHz en configuración quad channel.

En cuanto al software, se utilizó el sistema operativo Nyarch Linux de 64 bits. El código fue compilado utilizando el compilador GCC versión 15.2.1, empleando el estándar C++17 y las banderas de compilación -O2 -Wall.

Respecto a los datasets, se utilizaron aquellos que vienen del archivo generador de la tarea. No obstante, por restricciones de tiempo de ejecución, no se consideraron los arreglos de tamaño 
$10^7$ ni las matrices de 1024×1024 debido a su elevado costo computacional.
